# NWChem
Sript for NWChem's automatic compilation on Centos 6.x &amp; 7.x

# Before compile
The required packages of MPI variable compilation are following

OpenMPI (I recommend OpenMPI 1.6.5)
Intel Compilers
Intel MKL
python-devel
gcc-gfortran
openblas-devel
openblas-serial64
scalapack-openmpi-devel
elpa-openmpi-devel
tcsh
openssh-clients
which
