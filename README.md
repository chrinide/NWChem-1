# NWChem
Sript for NWChem's automatic compilation on Centos 6.x &amp; 7.x

# Before compile
The required packages of MPI variable compilation are following <br />
* OpenMPI (recommend OpenMPI 1.6.5) <br />
* Intel Compilers <br />
* Intel MKL <br />
* python-devel <br />
* gcc-gfortran <br />
* openblas-devel <br />
* openblas-serial64 <br />
* scalapack-openmpi-devel <br />
* elpa-openmpi-devel <br />
* tcsh <br />
* openssh-clients <br />
* which
